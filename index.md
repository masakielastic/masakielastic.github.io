# ようこそ

このサイトは、GitHub Pages と Jekyll を使って作成した個人ホームページのサンプルです。

Markdown で本文を書き、共通レイアウトで見た目を整えることで、少ないファイル数でも管理しやすいサイトを作れます。

  <section class="card">

      <li>自己紹介を載せる</li>
      <li>記事やメモを公開する</li>
      <li>GitHub や外部サービスへの導線を置く</li>
    </ul>
  </section>

  <section class="card">
    <h2>作り方のポイント</h2>
    <ul>

      <li>見た目はレイアウトにまとめる</li>
      <li>メニューは include で共通化する</li>
    </ul>
  </section>
</div>
